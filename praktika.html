Размер пакета фотографий

Цветной прибор формирует снимки размером 1024×768 пикселей с палитрой из 4096 цветов и упаковывает их пакетами по 256 снимков. Определите размер одного пакета в мегабайтах (целое число).

Пример алгоритма на JavaScript


// 1) Битность одного пикселя:
const bitsPerPixel = Math.log2(4096); // 12 бит

// 2) Размер одного снимка в байтах:
const pixels = 1024 * 768;
const bytesPerImage = pixels * bitsPerPixel / 8;

// 3) Размер пакета из 256 снимков в байтах:
const imagesInPacket = 256;
const totalBytes = bytesPerImage * imagesInPacket;

// 4) Перевод в мегабайты (1 МБ = 2^20 байт)
const MB = 2**20;
const totalMB = totalBytes / MB;

console.log("Размер пакета (МБ):", Math.floor(totalMB)); // 288
