Практическое задание

Цветной прибор фотографирует изображения размером 1024×768 пикселей с палитрой из 4096 цветов и формирует пакеты по 256 снимков. Определите размер одного пакета фотографий в мегабайтах (целое число).

Решение на Python

# Параметры
width, height = 1024, 768
colors = 4096
photos_per_packet = 256

# 4096 цветов => 12 бит на пиксель
bits_per_pixel = colors.bit_length() - 1  # или int(math.log2(colors))
# вычисляем размер одного фото в байтах
bytes_per_photo = width * height * bits_per_pixel // 8
# размер пакета
total_bytes = bytes_per_photo * photos_per_packet
# перевод в мегабайты (1 МБ = 2^20 байт)
megabytes = total_bytes // (2**20)
print(megabytes)  # целое число мегабайт
