Практическое задание

Игра «Камни»: у игроков есть куча. Петя начинает. За ход можно либо добавить 1 камень, либо удвоить количество камней в куче. Игра заканчивается, когда камней становится ≥ 129; победитель – сделавший последний ход. Пусть в начале было S камней (1 ≤ S ≤ 128). Найдите минимальное значение S, при котором одновременно выполняются: у Вани есть стратегия выигрыша за первый или второй ход (при любых ходах Пети), но у Вани нет стратегии гарантированного выигрыша за первый ход.

Решение на Python

def moves(x):
    return [x + 1, x * 2]

def game(s):
    if s >= 129:
        return 'П1'
    if any(game(m) == 'П1' for m in moves(s)):
        return 'В1'
    if all(game(m) == 'В1' for m in moves(s)):
        return 'П2'
    if any(game(m) == 'П2' for m in moves(s)):
        return 'В2'
    return 'Не выиграет'

for s in range(1, 129):
    if game(s) == 'В2':
        if all(game(m) != 'В1' for m in moves(s)):
            print(s)
            break

Этот код ищет минимальное значение S, при котором Ваня выигрывает за первый или второй ход, но не может выиграть сразу первым ходом.
