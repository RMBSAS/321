Программирование: циклы и условия

Описание: Задание связано с анализом кода: чаще всего это фрагмент на Pascal, Python или C++ с циклами и условиями. Нужно определить, какой результат вернёт программа, как изменяются переменные, сколько раз выполняются действия.

Навыки: Умение вручную выполнять трассировку кода: пошагово вычислять значения переменных, особенно внутри вложенных for, while и условий if. Важно знать правила работы логики и арифметики в выбранном языке.

Пример

count := 0
for i := 1 to 10 do
  if i mod 3 = 0 then
    count := count + 1
writeln(count)

Программа выводит 3, потому что делится на 3 только 3, 6, 9.

Решения

  • Начните с анализа внешнего цикла: сколько итераций он делает?
  • На каждой итерации проверьте условия if.
  • Следите за изменениями переменных: накапливаются ли значения (например, счётчик, максимум и т.д.).
  • После завершения цикла проверьте, что именно выводится или возвращается.

Советы и ошибки

  • Внимательно следите за диапазонами for — откуда начинается и где заканчивается цикл (включительно или нет).
  • Не пропускайте действия внутри if, особенно если вложены циклы.
  • Обязательно проверьте инициализацию переменных до начала цикла (например, count := 0).
  • Для сложных случаев удобно составлять таблицу значений по ходу итераций.

Полезные конструкции

  • mod — остаток от деления
  • div — целочисленное деление
  • Проверка кратности: if x mod d = 0
  • Счётчик: count := count + 1
  • Максимум: if x > max then max := x
