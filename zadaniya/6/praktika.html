Практическое задание

Исполнитель «Черепаха» действует на плоскости с декартовой системой координат. В начальный момент черепаха в начале координат, голова направлена вдоль оси y, хвост опущен (рисует линии). Даны две команды: «Вперёд n» (перейти на n вперед) и «Направо m» (повернуться на m градусов по часовой стрелке). Запись Повтори 7 [Вперёд 10 Направо 120] означает повтор последовательности внутри скобок 7 раз. Выполняется алгоритм:

Повтори 7 [Вперёд 10 Направо 120]

Определите, сколько точек с целочисленными координатами будут находиться внутри области, ограниченной линией, заданной этим алгоритмом (точки на границе не считаются).

Решение на Python

import math
from shapely.geometry import Polygon, Point

# Генерируем вершины многоугольника
x, y = 0.0, 0.0
angle = 90  # начальное направление вдоль оси y
vertices = []

for _ in range(7):
    # Вперёд 10
    rad = math.radians(angle)
    x += 10 * math.cos(rad)
    y += 10 * math.sin(rad)
    vertices.append((x, y))
    # Направо 120
    angle -= 120

# Закрываем полигон
poly = Polygon(vertices)

# Находим границы для обхода точек
minx, miny, maxx, maxy = map(math.floor, poly.bounds)

count = 0
for xi in range(minx+1, maxx):
    for yi in range(miny+1, maxy):
        if poly.contains(Point(xi, yi)):
            count += 1

print(f"Число целочисленных точек внутри: {count}")
