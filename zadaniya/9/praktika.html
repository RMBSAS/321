Практическое задание

Сколько существует восьмеричных пятизначных чисел, не содержащих в записи цифру 1, в которых все цифры различны и никакие две чётные или две нечётные цифры не стоят рядом? (Ответ – одно число без пояснений).

Решение на Python

count = 0
# Множества чётных и нечётных цифр в восьмеричной системе
even = {'0','2','4','6'}
odd = {'3','5','7'}  # исключаем '1'

for num in range(0o10000, 0o100000):  # от 8^4 до 8^5-1
    s = format(num, 'o')
    # Проверка длины и отсутствия '1'
    if len(s) != 5 or '1' in s:
        continue
    # Все цифры должны быть различны
    if len(set(s)) != 5:
        continue
    # Проверка, что нет двух подряд чётных или нечётных
    valid = True
    for i in range(4):
        if (s[i] in even and s[i+1] in even) or (s[i] in odd and s[i+1] in odd):
            valid = False
            break
    if valid:
        count += 1

print(count)  # Ответ задачи
