Практическое задание

Исполнитель «Редактор» преобразует строку цифр командами:

  • заменить(v,w) – заменить первое слева вхождение подстроки v на w;
  • нашлось(v) – проверяет наличие подстроки v в строке.

Дана программа:

НАЧАЛО
ПОКА нашлось(52) ИЛИ нашлось(2222) ИЛИ нашлось(1122)
  ЕСЛИ нашлось(52)    ТО заменить(52, 11)  КОНЕЦ ЕСЛИ
  ЕСЛИ нашлось(2222) ТО заменить(2222, 5) КОНЕЦ ЕСЛИ
  ЕСЛИ нашлось(1122) ТО заменить(1122, 25) КОНЕЦ ЕСЛИ
КОНЕЦ ПОКА
КОНЕЦ

На вход поступает строка, начинающаяся с «5», а затем n цифр «2» (3 < n < 10000). Определите наибольшее n, при котором сумма цифр результирующей строки равна 64.

Решение на Python

def transform(s):
    while '52' in s or '2222' in s or '1122' in s:
        if '52' in s:
            s = s.replace('52', '11', 1)
        if '2222' in s:
            s = s.replace('2222', '5', 1)
        if '1122' in s:
            s = s.replace('1122', '25', 1)
    return s

# Поиск максимального n
def sum_digits(s):
    return sum(int(c) for c in s)

max_n = 0
for n in range(4, 10000):
    inp = '5' + '2' * n
    res = transform(inp)
    if sum_digits(res) == 64:
        max_n = n
print(max_n)  # ответ задачи
