Практическое задание

В текстовом файле содержатся символы T, U, V, W, X, Y, Z. Определите максимальную длину (количество подряд) фрагмента, в котором буква T встречается ровно 100 раз. Для выполнения этого задания необходимо написать программу; в ответ запишите только найденное число.

Решение на Python

def max_length_with_100_Ts(s):
    left = 0
    count_T = 0
    max_len = 0
    for right in range(len(s)):
        if s[right] == 'T':
            count_T += 1
        while count_T > 100:
            if s[left] == 'T':
                count_T -= 1
            left += 1
        if count_T == 100:
            max_len = max(max_len, right - left + 1)
    return max_len

# Пример чтения из файла:
with open("24.txt") as f:
    data = f.read().strip()
    print(max_length_with_100_Ts(data))

Эта программа находит максимальную длину фрагмента строки, содержащего ровно 100 букв T. Она эффективно использует скользящее окно (двойной указатель).
