Обработка символьной информации

Описание: Необходимо проанализировать строку текста и выполнить операцию — подсчёт символов, изменение строки, поиск подстроки, фильтрация по условию и т.д. Чаще всего — пройтись по строке в цикле и сформировать результат по условию.

Навыки: Знание базовых операций над строками:

  • доступ к символам строки по индексу;
  • проверка типа символа (буква, цифра и т.д.);
  • поиск/замена символов или подстрок;
  • формирование новой строки на основе условия;
  • работа с функциями len(), isdigit(), lower(), replace() и др.

Пример

# Python
s = "AB1C2D3"
summa = 0
for c in s:
    if c.isdigit():
        summa += int(c)
print(summa)

Результат: программа находит сумму цифр: 1 + 2 + 3 = 6.

Решения

  • Создаём переменную-накопитель (строку, число, список и т.п.);
  • Проходим по каждому символу строки в цикле for;
  • Проверяем условие (например, if c in 'ABC' или c == '.');
  • Изменяем результат (добавляем к строке, считаем, пропускаем и т.д.);
  • Выводим результат.

Советы и ошибки

  • Строки индексируются с 0 (в Python) — не перепутайте при доступе s[0];
  • Для нечувствительности к регистру можно использовать s.lower() или s.upper() перед проверкой;
  • Для удаления символов используйте replace() или формируйте новую строку без них;
  • Проверяйте isdigit() перед приведением к числу — иначе может быть ошибка.

Полезные функции

  • c.isdigit() — проверка: символ — цифра;
  • c.isalpha() — символ — буква;
  • s.find('abc') — позиция подстроки или -1;
  • s.replace('A','') — удалить все 'A';
  • len(s) — длина строки.
