Практическое задание

Игра «Камни»: у игроков есть куча. Петя начинает. За ход можно либо добавить 1 камень, либо удвоить количество камней в куче. Игра заканчивается, когда камней становится ≥ 129; победитель – сделавший последний ход. Пусть в начале было S камней (1 ≤ S ≤ 128). Найдите минимальное S, при котором Петя не может выиграть за один ход, но при любом ходе Пети Ваня выигрывает своим первым ходом.

Решение на Python

def can_win(s):
    return s + 1 >= 129 or s * 2 >= 129

# Петя не выигрывает за 1 ход, но любой его ход приводит к победе Вани
for s in range(1, 129):
    if not can_win(s):
        next_moves = [s + 1, s * 2]
        if all(can_win(move) for move in next_moves):
            print("Ответ:", s)
            break

В этом коде мы проверяем, при каком минимальном S Петя не выигрывает сразу, но любое его действие (добавить 1 или удвоить) приводит к выигрышной позиции для Вани.
