Практическое задание

Найдите наименьшее целое неотрицательное число A, при котором выражение

(x + 2y < A) ∨ (y > x) ∨ (x > 60)

тождественно истинно для всех целых x, y ≥ 0.

Решение на Python

def is_true_for_all(A):
    for x in range(0, 61):  # при x>60 (x>60) истина, проверяем x<=60
        for y in range(0, x+1):  # при y>x (y>x) истина, проверяем y<=x
            if not ((x + 2*y < A) or (y > x) or (x > 60)):
                return False
    return True

# Поиск минимального A
a = 0
while True:
    if is_true_for_all(a):
        print(a)
        break
    a += 1  # следующий кандидат
