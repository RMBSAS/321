Практическое задание

Игра «Камни»: у игроков есть куча. Петя начинает. За ход можно либо добавить 1 камень, либо удвоить количество камней в куче. Игра заканчивается, когда камней становится ≥ 129; победитель – сделавший последний ход. Пусть в начале было S камней (1 ≤ S ≤ 128). Найдите два наименьших значения S, при которых у Пети есть выигрышная стратегия при одновременном выполнении: Петя не может выиграть за один ход, но Петя может выиграть вторым ходом независимо от ходов Вани.

Решение на Python

def can_win(s):
    return s + 1 >= 129 or s * 2 >= 129

def petya_second_win(s):
    if can_win(s):
        return False
    next_moves = [s + 1, s * 2]
    for move in next_moves:
        # Ваня ходит из позиции move
        vanya_moves = [move + 1, move * 2]
        if all(can_win(vm) for vm in vanya_moves):
            continue
        break
    else:
        return True
    return False

count = 0
for s in range(1, 129):
    if petya_second_win(s):
        print(s, end=' ')
        count += 1
        if count == 2:
            break

Этот код ищет первые два значения S, при которых Петя не выигрывает за один ход, но может гарантированно выиграть вторым ходом вне зависимости от действий Вани.
