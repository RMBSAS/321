Практическое задание

На вход алгоритма подаётся натуральное число N. Алгоритм строит новое число R так:

  1. Строится двоичная запись N;
  2. Если N делится на 3, то к двоичной записи слева приписывается «1», а справа – три последних цифры «111»;
  3. Если N не делится на 3, то остаток от деления на 3 умножается на 3, переводится в двоичную систему и дописывается в конец.

Полученная строка – это двоичная запись числа R. Укажите минимальное число R>151, которое может получиться в результате выполнения алгоритма.

Решение на Python

def build_R(N):
    b = bin(N)[2:]
    if N % 3 == 0:
        suffix = b[-3:]
        new_bin = '1' + b + suffix
    else:
        rem = (N % 3) * 3
        suffix = bin(rem)[2:]
        new_bin = b + suffix
    return int(new_bin, 2)

# Ищем минимальное R > 151
min_R = None
for N in range(1, 1000):  # граница поиска достаточно велика
    R = build_R(N)
    if R > 151:
        if min_R is None or R < min_R:
            min_R = R

print(f"Минимальное R > 151: {min_R}")
