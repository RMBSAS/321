Практическое задание

В файле дана последовательность натуральных чис (каждое ≤ 100000). Определите два значения:

  1. количество троек подряд идущих элементов, в которых ровно два из трёх – трёхзначные числа, а сумма троек не превышает максимального числа последовательности, оканчивающегося на 13;
  2. максимальную из сумм таких троек.

Запишите два числа: количество и найденную максимальную сумму.

Решение на Python

# Читаем числа из файла
data = [int(line) for line in open('data17.txt')]

# Находим максимальное число, оканчивающееся на 13
max_end13 = max(x for x in data if x % 100 == 13)

count = 0
max_sum = 0
# Перебираем все тройки подряд
for i in range(len(data) - 2):
    triple = data[i:i+3]
    # ровно два трёхзначных
    three_digit = sum(100 <= x <= 999 for x in triple)
    s = sum(triple)
    if three_digit == 2 and s <= max_end13:
        count += 1
        max_sum = max(max_sum, s)

print(count, max_sum)  # Вывод: количество троек и максимальная сумма
