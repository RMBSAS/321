Практика — Пары с делимостью на 3

На главную | Теория

В файле содержится последовательность целых чисел. Элементы могут принимать значения от −10 000 до 10 000 включительно. Определите и запишите в ответе:

  1. количество пар подряд идущих элементов, в которых хотя бы одно число делится на 3;
  2. максимальную из сумм элементов таких пар.

Например, для последовательности 6; 2; 9; –3; 6 ответ — 4 11.

Пример решения на JavaScript


// Читаем массив nums из файла
// (для примера возьмём встроенный массив)
const nums = [6, 2, 9, -3, 6];

let count = 0;
let maxSum = -Infinity;

// Перебираем пары подряд идущих элементов
for (let i = 0; i < nums.length - 1; i++) {
  const a = nums[i], b = nums[i+1];
  if (a % 3 === 0 || b % 3 === 0) {
    count++;
    const s = a + b;
    if (s > maxSum) maxSum = s;
  }
}

console.log(count, maxSum); // 4 11
