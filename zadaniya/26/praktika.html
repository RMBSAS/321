Практическое задание

В файле содержится информация о мероприятиях: в каждой заявке – время начала и окончания. Если время начала одного мероприятия меньше, чем окончание другого, их нельзя оба провести. Если время окончания одного совпадает с началом другого, провести можно оба. Определите максимальное число мероприятий, которые можно провести, и при этом максимальный возможный перерыв между двумя последними мероприятиями (в минутах). В ответе запишите два числа: сначала максимальное число мероприятий, затем максимальный перерыв.

Решение на Python

with open("26.txt") as f:
    n = int(f.readline())
    events = [tuple(map(int, line.split())) for line in f]

events.sort(key=lambda x: x[1])
result = []
last_end = -1

for start, end in events:
    if start >= last_end:
        result.append((start, end))
        last_end = end

max_gap = 0
for i in range(1, len(result)):
    gap = result[i][0] - result[i-1][1]
    max_gap = max(max_gap, gap)

print(len(result), max_gap)

Алгоритм реализует жадный выбор мероприятий по окончанию и затем ищет максимальный перерыв между двумя последними из выбранных мероприятий.
