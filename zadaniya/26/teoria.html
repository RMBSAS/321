Обработка чисел с сортировкой

Описание: Необходимо обработать числовой список, предварительно отсортировав его. После сортировки нужно найти нужный элемент (второй минимальный, второй максимальный и т.п.) или посчитать разности между соседними элементами.

Навыки:

  • Работа со списками чисел;
  • Использование сортировки (обычно sort());
  • Доступ к элементам по индексам (после сортировки);
  • Перебор упорядоченного массива в цикле.

Примеры задач

  • Отсортировать последовательность и найти второй по величине элемент.
  • После сортировки найти максимальную разность между соседними элементами.
  • Определить минимальное расстояние между элементами массива.

Пример кода

Дан список чисел. Найти максимальную разность между соседними после сортировки:

# Python
numbers = [13, 2, 9, 25, 18]
numbers.sort()  # Сортировка по возрастанию
max_diff = 0

for i in range(1, len(numbers)):
    diff = numbers[i] - numbers[i - 1]
    if diff > max_diff:
        max_diff = diff

print("Максимальная разность:", max_diff)

Полезные функции (Python)

  • list.sort() — сортирует список по возрастанию;
  • sorted(list) — возвращает отсортированную копию списка;
  • list[::-1] — переворот списка (для убывания);
  • range(1, len(list)) — перебор с доступом к соседним элементам.

Советы и частые ошибки

  • После сортировки первый элемент — минимальный, последний — максимальный;
  • Второй по величине элемент — list[-2] (при сортировке по возрастанию);
  • Не забудьте, что индексация начинается с 0 в Python;
  • Внимательно относитесь к типу сортировки: по возрастанию или по убыванию.
