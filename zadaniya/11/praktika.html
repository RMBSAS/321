Объём памяти для идентификаторов

В системе каждому объекту присваивается идентификатор длиной 60 символов (цифры + спецсимволы, всего 250 вариантов). Кодирование каждого символа – 11 бит. Для хранения каждого идентификатора выделяется целое число байт (одинаково для всех). Определите объём памяти в Кбайтах, необходимый для хранения 65 536 таких идентификаторов.

Пример алгоритма на JavaScript


// 1) Сколько бит нужно на один идентификатор:
const bitsPerSymbol = 11;
const length = 60;
const bitsPerId = bitsPerSymbol * length; // 660 бит

// 2) Переводим в целые байты:
const bytesPerId = Math.ceil(bitsPerId / 8); // 83 байта

// 3) Всего идентификаторов:
const countIds = 65536;

// 4) Общий объём в байтах:
const totalBytes = bytesPerId * countIds;

// 5) Переводим в килобайты (1 Кбайт = 1024 байт) и округляем вверх:
const totalKB = Math.ceil(totalBytes / 1024);

console.log("Объём (Кбайт):", totalKB); // 5314
