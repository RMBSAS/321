Практическое задание

В компьютерной системе каждому объекту присваивается идентификатор из 60 символов (только цифры и специальные символы алфавита). Для хранения каждого идентификатора используется одинаковое целое число байт, при этом применяется посимвольное кодирование (для каждого символа 11 бит). Определите объём памяти (в Кбайтах) для хранения 65 536 таких идентификаторов.

Решение на Python

import math

# Параметры
symbols = 60            # количество символов в идентификаторе
bits_per_symbol = 11    # бит на символ
count_ids = 65536       # число идентификаторов

# Вычисляем биты и байты на один идентификатор
total_bits = symbols * bits_per_symbol
bytes_per_id = math.ceil(total_bits / 8)

# Общий объём в байтах
total_bytes = bytes_per_id * count_ids

# Перевод в Кбайты (1 KB = 1024 байта)
total_kb = total_bytes // 1024

print(total_kb)  # Объём памяти в Кбайтах
