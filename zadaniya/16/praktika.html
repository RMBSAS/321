Рекурсивная функция F(n)

Функция F(n) задана рекурсивно:

  • Если n > 2024, то F(n) = n;
  • Если n ≤ 2024, то F(n) = n × F(n+1).
Найдите значение выражения F(2022) / F(2024).

Пример рассуждения и расчёта на JavaScript


// По определению:
// F(2025) = 2025  (так как 2025 > 2024)
// F(2024) = 2024 * F(2025) = 2024 * 2025
// F(2023) = 2023 * F(2024) = 2023 * 2024 * 2025
// F(2022) = 2022 * F(2023) = 2022 * 2023 * 2024 * 2025

// Тогда:
const num = 2022n * 2023n * 2024n * 2025n;
const den = 2024n * 2025n;
const result = num / den;  // 2022 * 2023
console.log("F(2022)/F(2024) =", result.toString()); // 409,  (2022*2023)=409,  (actual)

Примечание: результат есть произведение 2022×2023.
