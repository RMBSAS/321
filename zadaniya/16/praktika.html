Практическое задание

Алгоритм вычисления функции F(n) задан рекурсивно:

  • F(n) = n, если n > 2024;
  • F(n) = n * F(n+1), если n ≤ 2024;

Найдите значение выражения F(2022) / F(2024).

Решение на Python

import math

def F(n):
    if n > 2024:
        return n
    return n * F(n+1)

# Однако можно вывести формулу:
# F(2022) = 2022 * 2023 * 2024 * F(2025)
# F(2024) = 2024 * F(2025)
# => F(2022)/F(2024) = 2022 * 2023

result = 2022 * 2023
print(result)  # Ответ задачи
