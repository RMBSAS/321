Обработка целочисленной информации (без сортировки)

Описание: Задание предполагает работу с числами: проверку их свойств, подсчёт, нахождение делителей, сумм цифр и пр. Обычно обрабатываются числа из диапазона с помощью одного или двух вложенных циклов.

Навыки: Работа с целыми числами:

  • перебор диапазона с помощью for или while;
  • нахождение делителей, суммы/произведения цифр, количества цифр;
  • условные конструкции if для фильтрации чисел;
  • вывод нужных чисел и/или подсчёт их количества.

Пример

Найти все числа от 126849 до 126871 с ровно четырьмя делителями и вывести их делители:

# Python
for n in range(126849, 126872):
    divisors = [d for d in range(1, n+1) if n % d == 0]
    if len(divisors) == 4:
        print(n, divisors)

Решение

  • Заводим цикл по всем числам из диапазона;
  • Для каждого числа проверяем нужное условие: кол-во делителей, сумма цифр и пр.;
  • Если условие выполняется — выводим число или изменяем счётчик/максимум и т.п.;
  • В конце — вывод результата (список чисел, их делителей, кол-во и пр.).

Полезные функции (Python)

  • range(start, end) — генерация диапазона чисел (включает start, но не end);
  • n % d == 0 — проверка, что d — делитель n;
  • sum(map(int, str(n))) — сумма цифр числа n;
  • len([...]) — количество элементов в списке (например, делителей).

Советы и ошибки

  • Не забудьте включить правую границу диапазона (например, range(a, b+1));
  • При поиске делителей используйте range(1, n+1), либо оптимизированный подход с sqrt(n);
  • Если работаете с цифрами — преобразуйте число в строку: str(n);
  • Внимательно проверяйте условия: == (равно), != (не равно), >, < и т.д.
