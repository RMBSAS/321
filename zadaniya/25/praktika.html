Практическое задание

Называется маской числа последовательность, в которой цифра «?» – любая одна цифра, «*» – любая последовательность цифр (в том числе пустая). Маске 1?2157*4 соответствуют все числа вида 1A2157B4 (например, 123215734 и 1202157004). Среди натуральных чисел, ≤ 1010, найдите все числа, соответствующие маске 1?2157*4 и делящиеся на 2024 без остатка. Запишите в первом столбце все найденные числа (по возрастанию), во втором – результаты деления этих чисел на 2024.

Решение на Python

def matches_mask(n):
    from fnmatch import fnmatch
    return fnmatch(str(n), '1?2157*4')

for i in range(10**9, 10**10):
    if matches_mask(i) and i % 2024 == 0:
        print(i, i // 2024)

В этом решении используется функция fnmatch для проверки соответствия чисел заданной маске. Перебор происходит в пределах до 1010, но с учётом производительности рекомендуется добавить оптимизацию — например, перебирать только числа, кратные 2024.
