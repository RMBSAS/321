Практическое задание

Часть 1. В сети с адресом 192.168.32.160 и маской 255.255.255.240 найдите, сколько в этой сети IP-адресов имеют чётную сумму единиц в двоичной записи адреса. Ответ – число.

Часть 2. (не решается в ЕГЭ) – дополнительное условие, не включается в таблицу ответов.

Решение на Python

import ipaddress

# Определяем сеть
net = ipaddress.ip_network('192.168.32.160/28', strict=False)

count_even = 0
# Перебираем все адреса внутри сети (включая network and broadcast)
for addr in net:
    # Считаем единицы в 32-битном представлении
    ones = bin(int(addr)).count('1')
    if ones % 2 == 0:
        count_even += 1

print(count_even)  # число адресов с чётной суммой единиц
