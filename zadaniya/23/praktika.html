Практическое задание

Исполнитель преобразует число на экране. У исполнителя есть команды: A – прибавить 1, B – умножить на 2, C – возвести в квадрат. Программа – последовательность команд. Сколько существует программ, которые при исходном числе 2 дают результат 20 и при этом траектория вычислений программы не содержит число 11? (Траектория – это последовательность значений после каждой команды).

Решение на Python

def count_programs(start, end, forbidden):
    count = 0
    def dfs(current):
        nonlocal count
        if current == end:
            count += 1
            return
        if current > end or current == forbidden:
            return
        dfs(current + 1)  # A
        dfs(current * 2)  # B
        dfs(current ** 2)  # C
    dfs(start)
    return count

print(count_programs(2, 20, 11))

Эта программа считает количество различных последовательностей команд A, B и C, которые превращают число 2 в 20 без прохождения через 11.
