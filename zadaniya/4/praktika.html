Практическое задание

По каналу связи передаются сообщения, содержащие восемь букв: А, Б, В, Г, Д, Е, Ж, З. Для передачи используется двоичный код, удовлетворяющий условию Фано. Известны кодовые слова для А, Б, В, Г, Д, Е (см. таблицу). Определите минимальное количество двоичных знаков, необходимое для кодирования двух оставшихся букв (Ж, З).

Решение на Python

# Длины кодовых слов для известных букв A, B, V, G, D, E
known_lengths = [2, 3, 3, 3, 4, 4]  # пример: A=00, B=010, V=011, G=100, D=1010, E=1011

# Условие Фано (непрефиксный): должно соблюдаться неравенство Крафта:
# sum(2^{-l_i}) <= 1

# Вычислим остаточную «ёмкость» для двух символов
remaining = 1 - sum(2**(-l) for l in known_lengths)
print(f"Остаточная ёмкость: {remaining}")

# Попробуем найти минимальную сумму длин l1 + l2, таких что 2^{-l1} + 2^{-l2} <= remaining
best = None
for l1 in range(1, 10):
    for l2 in range(1, 10):
        if 2**(-l1) + 2**(-l2) <= remaining:
            total = l1 + l2
            if best is None or total < best[0]:
                best = (total, l1, l2)

print(f"Минимальная сумма длин: {best[0]} (l1={best[1]}, l2={best[2]})")
