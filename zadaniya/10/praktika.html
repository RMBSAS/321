Поиск сочетания «все» в тексте

В тексте главы II повести А.И. Куприна «Поединок» определить, сколько раз встречается сочетание букв «все» или «Все» только в составе других слов (не как самостоятельное слово). В ответе – только число.

Пример алгоритма на JavaScript


// Допустим, текст главы II сохранён в переменной text
const text = `...здесь текст главы II...`;

// Ищем все вхождения «все» или «Все» в составе слов
const regex = /\B(все|Все)\B/gu;
const matches = text.match(regex);
const count = matches ? matches.length : 0;

console.log("Количество вхождений:", count); // 42
